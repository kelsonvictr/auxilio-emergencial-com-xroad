Lists

CTRL+NUMPAD*transform paragraph to numbered list (<ol><li>)on paragraph
CTRL+NUMPAD-transform paragraph to bullet list (<ul><li>)on paragraph
CTRL+NUMPAD/transform paragraph to definition list (<dl><dt>)on paragraph
CTRL+NUMPAD+increase indentation of list item - create sub-liston list item
CTRL+NUMPAD-decrease indentation of list item - sub-list item to parent liston sub-list item
